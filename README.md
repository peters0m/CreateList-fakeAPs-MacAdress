# How to use

`$ git clone https://github.com/WILLYPETERSON11/mkfakeAPs`

`$ cd mkfakeAPs`
 
`$ do chmod +x mkfakeAPs.sh`

`$ ./mkfakeAPs -h`

# Example

`$ ./mkfakeAPs -r 10 test`

`$ sudo airmon-ng start [interface]`

I used WLO1 on the network interface

`$ sudo mdk3 b -v listfakeAPs.txt`

Check wifi list.
